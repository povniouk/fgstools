SPI Consistency Checker — Coming Soon

Ingests weekly SPI Excel exports, stores them in the database, and surfaces a delta report: new tags, changed fields, missing mandatory data. Waiting for complete data from HSED HOC and BoOC.

C&E vs Spec Checker — Coming Soon

Verifies cause-effect pairs in the C&E matrix against specification requirements. Waiting for the first C&E draft release.

Service

The app runs as a systemd service (fgstools) on the fgstools LXC. It starts automatically on boot and restarts on crash.

Restart and update controls will be added here once the admin API is wired up.

Logs

Use the Output panel at the bottom of the screen to view live app logs. On the LXC, full logs are available via: journalctl -u fgstools -f
